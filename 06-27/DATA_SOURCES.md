# Data sources

## 各鄉鎮市區人口密度.csv

- Dataset: 各鄉鎮市區人口密度
- Data year: 民國 106 年 (2017)
- Provider: 內政部戶政司
- Official source: https://data.gov.tw/dataset/8410
- License: 政府資料開放授權條款第 1 版

This local CSV is used as a learning dataset for the exercises in this directory.
It contains aggregate township and city-district statistics, not individual-level
personal data. Keep the dataset name, source URL, and license notice when it is
redistributed or adapted.
